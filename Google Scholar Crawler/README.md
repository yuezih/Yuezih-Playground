# Google Scholar Crawler
爬取 Google Scholar 某一作者的所有论文的共同作者，以分析论文合作情况。  
`main.py` 中使用的是 UESTC 某牛导的 Google Scholar 主页的数据，如需更换其他老师，替换 `urls` 列表即可，可在浏览器 `开发者工具栏` -> `网络` 中获取。示例结果如下：
